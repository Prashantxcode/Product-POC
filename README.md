# Product POC

A simple Product Listing Application built using HTML, CSS and JavaScript.

## Features

- Product Listing
- Product Details Page
- Search Products
- Sort Products
- Wishlist Functionality
- Cart Functionality
- Responsive UI

## Technologies Used

- HTML
- CSS
- JavaScript
- DummyJSON API

## Project Structure

Product POC/
├── index.html
├── product.html
├── cart.html
├── wishlist.html
├── styles.css
├── script.js
├── product.js
├── cart.js
└── wishlist.js

## API Used

https://dummyjson.com/products

## How to Run

1. Download or Clone the repository
2. Open index.html in browser
3. Explore products
